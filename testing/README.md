# Recipe App - Testing

## Description

This is used for testing all the services. It should not be run manually!

## Getting Started

This service should be very easy to get running! Follow the instructions below

## Dependencies

* Node / NPM

### Running the program

1. Install dependencies
    ```
    npm install
    ```

2. Start program
    ```
    cd ../
    sh scripts/run_test.sh
    ```
